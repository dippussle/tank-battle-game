# Tank Trouble Clone

A classic web-based tank battle game built with HTML5 Canvas and Vanilla JavaScript.

## Features

- **2-4 Local Players**: Battle with your friends on the same computer.
- **Random Maze Geometry**: Every round features a unique, procedurally generated maze.
- **Dynamic Bullet Physics**: Bullets bounce off walls, adding a layer of strategy (and danger!).
- **Clean Aesthetic**: Minimalist, top-down visual style inspired by classic tank games.

## Controls

### Player 1
- **Move**: WASD
- **Shoot**: Q

### Player 2
- **Move**: Arrow Keys
- **Shoot**: M

### Player 3
- **Move**: IJKL
- **Shoot**: U

### Player 4
- **Move**: 8456 (Numpad)
- **Shoot**: 0 (Numpad)

## Getting Started

Simply open `index.html` in your web browser to start playing!

## Development

- `index.html`: Main game structure.
- `style.css`: Visual styling.
- `script.js`: Core game logic, maze generation, and physics.
